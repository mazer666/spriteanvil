## Interface Overview

The desktop interface is divided into four main areas:

- **Left**: Tool Rail (Tools, Shapes, Selection)
- **Center**: Canvas Stage (Your workspace)
- **Right**: Properties Panel (Layers, Palette, Transforms, Adjustments)
- **Bottom**: Timeline (Animation frames)

## Tools

Shortcuts are shown in parentheses `( )`.

### Drawing

- **Pen (B)**: Standard single-pixel brush. Hard edges.
- **Eraser (E)**: Remove pixels.
- **Fill (F)**: Flood fill an area of similar color. Adjust 'Tolerance' in settings for gradients.
- **Line (L)**: Draw perfect straight lines.
- **Smudge (S)**: Drag to smear existing colors on the canvas, creating blending effects.

### Shapes

- **Rectangle (R) / Circle (C)**: Draw outlines.
- **Filled Shapes (Shift+R / Shift+C)**: Draw solid shapes.

### Selection

- **Rectangle (M) / Ellipse (Shift+M)**: Select geometric areas.
- **Lasso (W)**: Freehand selection.
- **Magic Wand**: Tap to select an area of similar color (uses Tolerance setting).
- **Move (V)**: Move the selected pixels.

## Layers

Located in the Right Panel (Desktop) or Layers Popover (Mobile).

- **eye icon**: Toggle visibility.
- **lock icon**: Prevent changes.
- **Blend Modes**: Change how the layer mixes with layers below (Multiply, Overlay, etc).

## Animation

Use the Timeline at the bottom.

- **Add Frame**: Create a new empty frame or duplicate the current one.
- **Onion Skinning**: See faint ghosts of previous/next frames to help animate smooth motion.
- **Play (Space)**: Preview your animation.

## Exporting

1. Click **Export** in the top bar (Cmd+E).
2. Choose **Spritesheet** (for games) or **GIF** (for sharing).
3. Configure columns/rows for spritesheets.
4. Click Download.
