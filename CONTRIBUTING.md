# Contributing to SpriteAnvil

Welcome! Whether you are a human or an AI assistant, this guide will help you contribute to SpriteAnvil.

## Getting Started
1. Read the [Code Style Guide](docs/CODE_STYLE.md).
2. Follow the [Common Tasks Guide](docs/COMMON_TASKS.md) for step-by-step instructions.
3. Check [ARCHITECTURE.md](docs/ARCHITECTURE.md) to understand the project structure.

## For AI Assistants
- **Context**: Read `llms.txt` first for a high-level overview.
- **Standards**: Follow the rules in `.cursorrules`.
- **Development**: Use the [AI Development Guide](docs/AI_DEVELOPMENT_GUIDE.md) for optimized prompt patterns.

## Development Workflow
1. `npm install`
2. `npm run dev`
3. Make changes following the modular rules (400-line React limit).
4. Run tests: `npm test` (if available).

## Reporting Issues
Use GitHub issues to report bugs or suggest features. Mention if the issue was identified by an AI scan.
